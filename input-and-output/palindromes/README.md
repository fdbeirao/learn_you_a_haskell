## Palindromes

Taken from [Learn you a Haskell](http://learnyouahaskell.com/input-and-output#files-and-streams)

## Execution without compilation

```
runhaskell ./palindromes
elephant
ABCBA
whatever
```

**`Ctrl+C` to exit**

## Compilation && Execution

```
ghc --make ./palindromes.hs; ./palindromes
elephant
ABCBA
whatever
```

**`Ctrl+C` to exit**
